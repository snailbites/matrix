Buttons

Buttons can be anchor tags or button tags.

Primary button

Primary buttons use primary action color and have rounded corners.

Block button

Block buttons take the full width of their container

Large button

These are larger versions typically used for submitting forms

Button with icon

Primary buttons can also have an icon as an extra affordance.

Secondary button

Secondary buttons denote secondary actions and typically accompany primary buttons.
